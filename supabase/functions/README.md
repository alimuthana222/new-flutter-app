# Supabase Edge Functions

This directory is for Supabase Edge Functions (Deno-based serverless functions).

## Creating a Function

To create a new Edge Function:

```bash
supabase functions new function-name
```

## Deploying Functions

To deploy functions to your Supabase project:

```bash
supabase functions deploy function-name
```

## Testing Locally

To test functions locally:

```bash
supabase functions serve function-name
```

## Example Functions

You can create functions for:
- Sending custom notifications
- Processing payments
- Scheduled tasks (cron jobs)
- Webhooks for external services
- Complex business logic that should run server-side

## Documentation

For more information, see the [Supabase Edge Functions documentation](https://supabase.com/docs/guides/functions).
